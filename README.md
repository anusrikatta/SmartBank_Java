# SmartBank – Console Banking System (Java)

SmartBank is a console-based banking management system developed using Java. The application allows users to create accounts, securely log in using a PIN, and perform banking operations such as deposit, withdrawal, and balance inquiry.

## 🚀 Features

* Account registration system
* Secure login using account number and PIN
* Deposit money
* Withdraw money
* Check account balance
* Transaction history with date and time
* Data persistence using file handling

## 🛠 Technologies Used

* Java
* Object-Oriented Programming (OOP)
* File Handling
* Collections (HashMap)
